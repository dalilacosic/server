Documentation


User api

@route /api/v1/register

@method POST

@params { username(not empty), email(required), password(not empty) }

@returns { msg }

@field-errors { status: 400, errors: Array }

@db-errors { status: 409, error: String }


@route /api/v1/authenticate

@method POST

@params { email(required), password(not empty) }

@returns { token: `${token}`, succes: Boolean }

@auth-errors { status: 401, error: String }


@route /api/v1/user/location

@method GET

@returns { username: String, coords: { longitude: Float, latitude: Float } }

@auth-errors { status: 401, errors: String }


Description: API used for getting current user location


@route /api/v1/user/all

@method GET

@returns Array({ user: {username: String, id: String}, coords: { longitude: Float, latitude: Float } })

@auth-errors { status: 401, errors: String }

@db-errors { status: 500, errors: String }


Description: API used to get all of users coordinates register in DB, except user requesting them


@route /api/v1/user/location

@method PUT

@params { longitude: Float, latitude: Float }

@returns { coords: { longitude: Float, latitude:Float }

@auth-errors { status: 401, errors: String }

@db-errors { status: 500, errors: String }


Description: API used to update user location



Message api

@route /api/v1/message/all

@method GET

@returns Array({ content: String, user: { username: String, _id: ObjectId } })

@auth-errors { status: 401, errors: String }

@db-errors { status: 500, errors: String }


Description: API used to get message history

@route /api/v1/message/create

@method GET

@params { content: String }

@returns { msg: String }

@auth-errors { status: 401, errors: String }

@db-errors { status: 500, errors: String }


Description: API used to save new message to DB
